# WeTalk (微通话) 每日自动签到与广告挂机助手

这是一个基于 **Python** 与 **ADB** 的 WeTalk 自动签到与看视频广告挂机助手。通过模拟人工点击和底层布局解析，实现稳定、高耐久的余额累积，支持跨品牌安卓设备。

---

## ✨ 核心特性

1. **底层布局级精准定位**：不依赖死板的坐标点击，通过解析手机 XML 视图树智能定位签到、视频及关闭按钮，完美适配各种屏幕分辨率。
2. **多厂商系统弹窗自动兼容**：自动识别并处理系统权限申请（如通知允许/取消等）及各类安全卫士拦截，防止挂机中断。
3. **智能广告关闭引擎**：自动识别 Google Ads 等主流广告平台的“跳过”、“关闭”及“X”按钮，保障顺畅流转。
4. **防早关警告保护**：当误触或广告商弹出“关闭将失去奖励”警告时，自动点击“继续播放”并启动 **22秒静默期**，保护账号安全，防风控。
5. **硬核主动自愈机制**：如果连续点击视频未成功触发广告，脚本会自动强杀应用 -> 切换手机飞行模式重置基站 IP（闪断 IP）-> 重新拉起应用自愈。
6. **双重状态分流**：区分冷启动推广弹窗与真实结算弹窗，确保成功次数统计绝对精准。

---

## 🛠️ 环境准备

1. **PC 端环境**：安装 Python 3.x，并确保已配置到环境变量中。
2. **ADB 驱动**：脚本默认使用 O+Connect 内置的 ADB 路径 `D:\O+Connect\daemon\bin\adb.exe`。如果使用其他投屏工具，可将 ADB 加入环境变量，脚本将自动回退使用 `adb` 命令。
3. **手机端设置**：
   - 打开**开发者选项**，开启 **USB 调试**。
   - **非常重要**：部分手机（如小米、iQOO、OPPO）需额外在开发者选项中开启 **“允许通过 ADB 模拟点击 / 安全设置”**，否则脚本只能读取界面无法模拟点击。
   - 保持手机连接电脑，开启“充电时屏幕不锁定”以便长时间挂机。

---

## 🚀 快速启动

1. 解压项目到本地目录。
2. 双击运行 `run.bat`，即可启动控制台仪表盘进行自动挂机。
3. 挂机过程中，在控制台按下 `Ctrl + C` 可安全退出。
